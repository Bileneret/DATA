# Практична робота №3
## Декомпозиція часових рядів та аналіз енергоспоживання (Варіант 14)

### [Код завдання](prac3.py)

#### Опис:
Програма `prac3.py` реалізує аналіз часових даних про споживання енергії за допомогою статистичного методу декомпозиції. Використовуючи інструмент `seasonal_decompose` з бібліотеки `statsmodels`, часовий ряд розбивається на три ключові складові: трендову, сезонну та залишкову (шум). Програма також оцінює точність прогнозування споживання на основі комбінації тренду та сезонності.

#### Особливості роботи:
##### - Декомпозиція ряду – розділення даних на довгостроковий тренд, циклічну сезонність та випадкові відхилення.
##### - Візуалізація компонентів – побудова детального багатопанельного графіка для аналізу кожної складової окремо.
##### - Оцінка точності – розрахунок метрик MAE та RMSE для перевірки відповідності реконструйованої моделі (тренд + сезонність) реальним даним.
