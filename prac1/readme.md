# Практична робота №1
## Оптимізація пам'яті та аналіз похибки типів даних (Варіант 14)

### [Код завдання](prac1.py)

#### Опис:
Програма `prac1.py` генерує великий синтетичний набір даних (100 мільйонів рядків) та демонструє методи оптимізації DataFrame у бібліотеці Pandas. Основна мета — порівняти використання оперативної пам'яті при 64-бітних (`float64`, `int64`) та 32-бітних (`float32`, `int32`) типах даних. Також програма розраховує похибку середнього значення колонки `price`, що виникає внаслідок втрати точності під час даункастингу (зменшення розрядності).

#### Особливості роботи:
##### - Генерація датасету – створення 100 млн записів (колонки: `id`, `price`, `quantity`, `discount`).
##### - Аналіз RAM – автоматичне обчислення обсягу пам'яті (в мегабайтах) до та після перетворення типів.
##### - Оцінка точності – розрахунок математичної похибки між значеннями `mean()` для `float64` та `float32`.
