# Практична робота №6
## Обробка та аналіз реальних даних з мобільних сенсорів

### [Код завдання](prac6-var1.py)

> **📌 Важливе пояснення щодо вибору варіанту:**
> Згідно з розподілом, мій варіант — **№14** (Дослідження атмосферного тиску). Однак, мій основний пристрій апаратно не оснащений датчиком барометра. Оскільки головною і найважливішою вимогою до цієї практичної роботи було **збирання та обробка реальних фізичних даних з сенсору (заборона на генерацію)**, я прийняв рішення виконати **Варіант №1** (Вимірювання прискорення). Це дозволило мені повноцінно продемонструвати навички роботи з "сирими" даними акселерометра за допомогою додатка Phyphox.

#### Опис:
Програма `prac6-var1.py` аналізує реальні дані лінійного прискорення, зібрані під час вільного руху. Скрипт не просто ділить запис за часом, а використовує динамічний алгоритм на основі ковзної дисперсії (Rolling Variance), який автоматично розпізнає патерни рухів та класифікує їх на чотири стани: спокій, спокійна хода, швидка хода та біг.

#### Особливості роботи:
##### - Робота з "сирими" даними – завантаження та парсинг реальних логів з акселерометра (формат CSV).
##### - Цифрова фільтрація – застосування фільтра Савицького-Голея (`savgol_filter`) для згладжування високочастотного апаратного шуму датчика.
##### - Feature Engineering – впровадження методу ковзного вікна (`rolling.var`) для автоматичної класифікації активності користувача незалежно від тривалості запису.
##### - Статистичний аналіз – обчислення відсотка часу перебування у кожному стані та визначення середнього прискорення для кожної фази руху.
